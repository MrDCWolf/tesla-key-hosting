Tesla Fleet API Public Key Hosting

This site hosts the public key required for Tesla Fleet API registration.

Public Key Location

The public key is available at:

https://[your-github-username].github.io/tesla-key-hosting/.well-known/appspecific/com.tesla.3p.public-key.pem

If you can see this page, your hosting setup is working correctly!

Next Steps

After confirming your public key is accessible at the URL above, complete the registration process with:

python3 tesla_register.py register --client-id YOUR_CLIENT_ID --client-secret YOUR_CLIENT_SECRET
